git init---> para iniciar el repositorio
git add---> anade un archivo o directorio al repositorio
git clone [direccionURL]---> clona la carpeta actual del proyecto
git rm---> borra un archivo
git mv [origen][destino]---> mueve un archivo
git status---> presenta el estado actual del arbol de trabajo en local y la captura
git diff---> muestra la diferencia del local y el staged
git diff [branch1][branch2]---> muestra la diferencia de dos rama
git log--->muestra el historial del commit
git log --oneline---> muestra cada commit en una linea
git reset -- hard codigo--->volvemos al commit indicado y borra todo lo demas
git reset HEAD [arvhico]---> para restaurar un archivo
git branch---> para saber las ramas del reposotorio
git branch---> [nombreRama] crea una rama
git merge [nombreRama]---> une una rama con la actual (se recomienda hacerlo desde la master)
git checkout [nombreRama]---> para cambiar de rama
git checkout -b [nombreRama]---> crea una rama y nos posiciona en ella
git branch -d [nombreRama]--->elimina una rama
